# Example - the heading elements

```
<body>
  <header>
    <title> page title </title>
    <p> subtitle </p>
  </header>
  ...
</body>
```

- compared with

```
<body>
  <h1> section title </h1>
</body>
```

- could be seen as

```
<body>
  <h1>
    <title> section title </title>
  </h1>
</body>
```

- could be extended as

```
<body>
  <h1>
    <title> section title </title>
    <p> subtitle </p>
  </h1>
</body>
```
