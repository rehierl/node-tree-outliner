# Stack of open sections

Traversing the relevant tree in order to produce an outline needs to (implicitly
or explicitly) create and update stacks of open sections.

```
Example 3 (E-3):
----------------

<h1>A</h1>
B
<h2>C</h2>
D
<h3>E</h3>
F
<h2>G</h2>
H
```
